# 🚀 Professional Portfolio - Habib Syuhada

A modern, professional portfolio website built with React and Vite, designed for a corporate and clean aesthetic.

## 🌐 Live Demo
Check out the live site here: [https://habibsyuhada.github.io/portfolio/](https://habibsyuhada.github.io/portfolio/)

## 🛠️ Tech Stack

- **Framework:** React 18+
- **Build Tool:** Vite
- **Styling:** CSS Modules (Scoped CSS)
- **Routing:** React Router DOM
- **Icons:** React Icons
- **Deployment:** GitHub Pages (via `docs` folder)

## ✨ Key Features

- **Professional Design:** Clean typography, structured layout, and corporate aesthetic.
- **Dark/Light Mode:** Seamless theme switching with local storage persistence.
- **Responsive:** Fully optimized for mobile, tablet, and desktop screens.
- **Smooth Navigation:** Scroll-spy integrated navbar with smooth scrolling behavior.
- **Easy Maintenance:** All content (projects, skills, personal info) is managed in a single `portfolio.json` file.

## 📂 Project Structure

```text
src/
├── assets/          # Images and static assets
├── components/      # UI components (Layout, Sections, UI)
├── data/            # Content management (JSON and mapping)
├── hooks/           # Custom React hooks (Theme, ScrollSpy)
├── styles/          # Global styles and theme variables
└── App.jsx          # Main application entry
```

## 🚀 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/habibsyuhada/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📦 Deployment

The project is configured to deploy to GitHub Pages using the `docs` folder:

```bash
npm run deploy
```
